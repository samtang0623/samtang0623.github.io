冤罪殺機 Tips

冤罪殺機2

本作的密碼除了第一個之外全部隨機，因此列出密碼位置。獲得密碼線索之後會直接顯示。

CH1

皇宮下方保險箱：451

CH2

督軍房間

  1. 打開保險箱旁邊的書，這是數字提示
  2. 同層房間辦公桌上的紙張就是密碼本體
    保險箱店
  3. 前台獲得收銀器密碼
  4. 打開店鋪後面的保險箱

CH3

斯瓦科保險箱

  1. 主線與斯瓦科醫生見面
  2. 獲得密碼就能打開了
    二樓保險箱
  3. 紅色桌子上的紙條獲得密碼
  4. 打開旁邊的保險箱

CH4

車站旁邊保險箱

  1. 車站小道上樓
  2. 小黑板位置打碎黑板上的玻璃片就是密碼
  3. 打開保險箱

CH5

血蠅保險箱

  1. 上岸往左走，可以根據塗鴉找到黑市
  2. 在黑市買下保險箱密碼
  3. 被大量白布包裹的就是血蠅房間

CH6

本章沒有保險箱，只有黑市密碼

  1. 黑市有賣骸骨，可以用這個找到黑市位置
  2. 短劍毀掉木板，弩箭射擊毀掉鐵門後的木板
  3. RAIN後面的畫圈就是黑市密碼

CH7

小房間

  1. 密碼箱藏著骸骨的房間外，穿越回過去進入桌子底下
  2. 桌子底下回到現代就能進入房間
  3. 在現代可以看到密碼箱第一位數字
  4. 後面從00到99去測試，很快就能打開

CH8

保險箱老闆

  1. 來到保險箱店，打開收銀機獲得老闆公寓鑰匙
  2. 老闆家有骸骨，不會太難找
  3. 用望遠鏡觀察保險箱對面的畫可以獲得密碼
  4. 打開保險箱
    大隊長保險箱
  5. 在冒牌公爵辦公室的桌上獲得密碼
  6. 從下方窗戶進入房間打開保險箱

CH9

再拿一次

  1. 最後一個保險箱就是遊戲中的第一個保險箱
  2. 密碼一樣固定是451
